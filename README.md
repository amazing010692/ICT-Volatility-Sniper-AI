## Signal Logic

The signal engine uses a **three-path approach**:

**Path 1 — Impulse Override (fastest):**
If a candle is abnormally large (displacement, impulse, or volume spike), the system fires immediately in that direction regardless of other factors. This catches the start of explosive moves.

**Path 2 — Momentum Phase Boost:**
The advanced momentum engine classifies the current phase. IGNITION and ACCELERATION phases get confidence boosts. CLIMAX and EXHAUSTION phases get penalized or killed entirely. This prevents chasing and ensures early entries.

**Path 3 — Confluence Scoring:**
Standard checklist-based scoring with aggressive thresholds. Momentum and liquidity sweeps are weighted highest. Structure disagreement only downgrades confidence (doesn't kill the signal).
